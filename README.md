# Innovator Founder Visa Landing Page

Single-page HTML landing page for an Innovator Founder Visa consultancy positioned as startup builders and visa strategists.

## Live Site

https://innovator-founder-visa.netlify.app/

## Overview

This project contains a responsive marketing page built for lead generation around UK Innovator Founder Visa services.

## Stack

- HTML
- Tailwind CSS via CDN
- Vanilla JavaScript

## Files

- `index.html` - main landing page
- `checklist.html` - printable client preparation checklist page
- `pricing.html` - printable detailed pricing guide page
- `az-process.html` - printable A-Z success roadmap page
- `basic-package.html` - printable detailed Basic package page
- `standard-package.html` - printable detailed Standard package page
- `premium-package.html` - printable detailed Premium package page
- `plan.md` - original build brief

## Features

- Sticky header with logo
- Dedicated printable checklist page with print-to-PDF support
- Dedicated printable pricing guide page with print-to-PDF support
- Dedicated printable A-Z process page with print-to-PDF support
- Dedicated printable package pages for Basic, Standard, and Premium
- Image-led hero section
- Services, pricing, process, testimonials, and FAQ sections
- Direct contact section with phone, Telegram, WhatsApp, and email
- Smooth scrolling and section reveal effects

## Run Locally

Open `index.html` directly in a browser, or serve the folder with a simple local server such as VS Code Live Server.
