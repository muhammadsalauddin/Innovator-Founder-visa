# Innovator Founder Visa Landing Page

Single-page HTML landing page for an Innovator Founder Visa consultancy positioned as startup builders and visa strategists.

## Overview

This project contains a responsive marketing page built for lead generation around UK Innovator Founder Visa services.

## Stack

- HTML
- Tailwind CSS via CDN
- Vanilla JavaScript

## Files

- `index.html` - main landing page
- `plan.md` - original build brief

## Features

- Sticky header with logo
- Image-led hero section
- Services, pricing, process, testimonials, and FAQ sections
- Direct contact section with phone, Telegram, WhatsApp, and email
- Smooth scrolling and section reveal effects

## Run Locally

Open `index.html` directly in a browser, or serve the folder with a simple local server such as VS Code Live Server.
